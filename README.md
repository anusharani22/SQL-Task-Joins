# Task 5: SQL Joins (Inner, Left, Right, Full)

## Objective
Learn to combine data from multiple tables using different join types.

## Tools Used
- DB Browser for SQLite / MySQL Workbench

## Steps Performed
1. Created two related tables: *Customers* and *Orders*.
2. Inserted sample data into both tables.
3. Wrote queries for:
   - INNER JOIN
   - LEFT JOIN
   - RIGHT JOIN
   - FULL OUTER JOIN
## Outcome
Gained mastery over merging data from multiple tables using different join types.
